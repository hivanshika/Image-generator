# AI Prompt to Image Generator

## Overview
This project converts a simple user idea into a professional AI-generated image. Users enter a plain-language description, and the application generates a high-quality image using the Hugging Face Inference API.

## Features
- Simple Streamlit interface
- Converts everyday ideas into AI-generated images
- Powered by Hugging Face Inference API
- Fast and easy to use

## Tech Stack
- Python
- Streamlit
- Hugging Face Inference API
- python-dotenv

## How to Run

1. Clone the repository.
2. Install the dependencies:

```bash
pip install -r requirements.txt
```

3. Create a `.env` file:

```env
HF_TOKEN=your_huggingface_token
```

4. Run the application:

```bash
streamlit run app.py
```

## Example Prompt

```
A poster for my juice shop's monsoon offer
```
